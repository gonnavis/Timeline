<img src="http://gonnavis.com/timeline/preview2.png">

<h3>简介</h3>
<p>自动排列，直观地显示各个历史时间段。</p> 

<h3>运行</h3>
<p>
  npm install<br>
  npm run dev<br>
  http://localhost:8080/<br>
</p>

<h3>编译</h3>
<p>
  npm run build<br>
</p>

<h3>计划开发功能</h3>
<p>对应时期的地图，完善标尺，时间段嵌套，自定义／上传时间段。 </p>
